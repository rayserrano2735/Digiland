üíôüíªü§ùRetryRSPython 1: Create a function called insertOrder() that will insert a number into a list such that the list will always be in order. You may assume that the elements of the input list are already in ascending order.Editpythondef insertOrder(lst, num):
    # Binary search to find insertion point
    left, right = 0, len(lst)
    
    while left < right:
        mid = (left + right) // 2
        if lst[mid] < num:
            left = mid + 1
        else:
            right = mid
    
    lst.insert(left, num)
    return lst
Or simpler but less efficient for large lists:
pythondef insertOrder(lst, num):
    lst.append(num)
    lst.sort()
    return lst
The binary search approach is O(log n) for finding position but O(n) for insertion. The append+sort is O(n log n) but cleaner code.
